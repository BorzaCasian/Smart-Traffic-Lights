Codul intitulat "semafor" este codul pentru semaforul inteligent.
Codul intitulat "cod_motor" este codul pentru masinutele controlate prin infra rosu.

Utilizarea celor doua coduri:
  Pasul 1:  se descarca platforma ArduinoIDE
  Pasul 2: se instaleaza librariile de la inceputul celor doua programe 
  Pasul 3: se selecteaza platfomra Arduino UNO ca si placa de dezvoltare
  Pasul 4: se alege port-ul dorit prin care ati conectat placa arduino la calculator
  Pasul 4: compilam codul
  Pasul 5: incarcam codul pe placa de dezvoltare Arduino
